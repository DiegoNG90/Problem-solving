Que es un STACK?

Ejemplo STACK OF BOOKS: el último libro es el último en entrar y el primero en agarrar/retirar.

Otro EJEMPLO podría ser el botón del navegador para volver atrás; vuelve a lo último que visitaste.

Supongamos que vistiamos 3 sitios:

  1. Facebook
  2. Google
  3. Wikipedia

Podemos asemejar al stack con una array. De pesta manera, ese array-stack estaría formado por stack.push (Facebook), stack.push (Google) y stack.push (Wikipedia).

Si nosotros queremos apretar el boton de atras/regresar, se va a generar un stack.push() -recordemos que el método push() retira el último elemento de una array, no se le pasan parametros-. De ésta manera, sacámos a Wikipedia de ésta array-stack y nos muestra la pagina de Google y así de nuevo -mismo proceso con Facebook

Funciones del stack

  • Push
  • Pop
  • Peek - Para mostrar el último (top) elemento del stack
  • Length / Size

Un buen ejercicio se puede observar en palindrome.js

function isPalindrome(word){ 
    const toLowerWord = word.toLowerCase(); 
    const letters = []; 
    let reverseWord = "";
    //Lleno la array vacia con cada letra de la string WORD que pasemos como parametro

    for (let i = 0; i < word.length; i++) { 
        letters.push(toLowerWord[i]); 
    }
    for (let i = 0; i < word.length; i++) { 
        reverseWord += letters.pop();      
    }

    console.log(toLowerWord);
    console.log(reverseWord);
    return (toLowerWord === reverseWord) ? console.log( `${word} is a palindrome` ): console.log( `${word} is not a palindrome`);
}

El objeto constructor de un Stack podria armarse de la siguiente manera

const Stack = function(){
    //constructor

    this.count = 0;
    this.storage = {};

    //Add a value onto the END of the stack 

    this.push = function(value){
        this.storage[this.count] = value;
        this.count++;
    }

    //Remove and return the value at the END of the stack

    this.pop = function(){
        if (this.count === 0){
            return undefined;
        }
        this.count--;
        let result = this.storage[this.count];
        delete this.storage[this.count];
        return result;
    }

    //Size or Length

    this.size = function(){
        return this.count;
    }

    //Peek returns the value at the end of the stack

    this.peek = function(){
        return this.storage[this.count-1];
    }
}
